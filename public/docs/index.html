    Introduction

    Welcome to the CIAS 3.0 (Computerized Intervention Authoring System)

    Base URL

    localhost:3000

    This describes the resources that make up the official cias-api.herokuapp.com REST API. In order to use resources, You have to obtain an appropriate token.

    Authentication

    Create session

    Resource

    localhost:3000/auth/sign_in

    URL parameters

    Parameter Description Required

    Query String

    Parameter Description Required

    HTTP request method

    POST

    HTTP headers

    Key Value
    Content-Type application/json; charset=utf-8

    HTTP body

    { "email": "email", "password": "password" }

    Note

    1. For every new HTTP request to another reosource, we use:
      1. New access-token retrieved from a HTTP response;
      2. Rest reciveced headers which wer recived during the creating the session without modifing them. Read more here.

    HTTP example request

    POST localhost:3000/auth/sign_in

    Content-Type: application/json; charset=utf-8

    {
  "email": "user@cias-api.herokuapp.com",
  "password": "password"
}

    HTTP response 200 OK

    Content-Type: application/json; charset=utf-8
access-token: xxxyyyzzz
token-type: Bearer
client: xxxyyyzzz
uid: user@cias-api.herokuapp.com

    {
  "data": {
    "id": 1,
    "deactivated": false,
    "email": "user@cias-api.herokuapp.com",
    "provider": "email",
    "roles": [
      "admin"
    ],
    "uid": "user@cias-api.herokuapp.com",
    "first_name": "admin",
    "last_name": "1",
    "username": "user",
    "allow_password_change": false
  }
}

    HTTP response 401 Unauthorized

    {
  "success": false,
  "errors": [
    "Invalid login credentials. Please try again."
  ]
}

    Errors

    The CIAS 3.0 uses the following error codes:

    Code Description
    400 Bad Request -- Your request is invalid.
    401 Unauthorized -- Your API key is wrong.
    403 Forbidden -- The requested resource is available only for users with different scope.
    404 Not Found -- The specified resource could not be found.
    405 Method Not Allowed -- You tried to access a resource with an invalid method.
    406 Not Acceptable -- You requested a format that isn't JSON.
    410 Gone -- The requested resource has been removed from our servers.
    418 I'm a teapot.
    429 Too Many Requests -- You're requesting too many resources.
    500 Internal Server Error -- We had a problem with our server. Please try again later.
    503 Service Unavailable -- We're temporarily offline for maintenance. Please try again later.
